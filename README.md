# 🚗 Limousine Booking System

A modern limousine booking web application built with **React + TypeScript + Vite**.  
This project allows users to search routes, select pickup/drop-off locations, calculate distance, and book limousine services with a smooth and responsive UI.

---

## ✨ Features

- 🗺️ Google Maps integration (Places Autocomplete & Directions)
- 📍 Pickup and Drop-off location search
- ➕ Add multiple stops (optional routing feature)
- ⏱️ Automatic distance and travel time calculation
- 🚘 One-way and hourly booking options
- 📱 Fully responsive UI for desktop and mobile
- ⚡ Fast development setup using Vite
- 🎨 Clean and modern UI design

---

## 🛠️ Tech Stack

- React (TypeScript)
- Vite
- Google Maps JavaScript API
- Places API
- Tailwind CSS (if applicable in your project)
- Axios (if used for API calls)

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/ellezaj/limousine-booking.git
cd limousine-booking
```

Install dependencies:

```bash
npm install
```

Create environment file:

```bash
VITE_GOOGLE_MAPS_API_KEY=your_api_key_here
```

Run the project:

```bash
npm run dev
```

Run the mock endpoint:

```bash
npm run mock
```
